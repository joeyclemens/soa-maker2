Getting Started

When you open the SoA_Template Access file, you'll be greeted. The program will load and await your input.
Once its loaded, pick a project and have fun.

Lets build a SoA

When you have selected a project, the fun can begin. Two selection boxes will appear: Select Area & Select Department. You may also notice the plethora of buttons that have suddenly appeared in the side bar.
This is the entire layout of the program and should hopefully be somewhat self-explanatory.

The buttons

  • The buttons will carry out the behind the scenes operations that make the magic happen.
  • "Create SoA". This button will create the SoA and export it into Excel. Use this when you're happy with the SoA you've created.
  • "Add Area to SoA". This button will add an area (The first selection box) to the SoA. Select the Area(s) you want to add amd then press this button. It will add entire Areas to the SoA.
  • If you need a more granular approach then these next few buttons will be of interest to you.
  • "Add Department to SoA". This button will add a Department (The second selection box) to the SoA. Select the Department(s) you want to add amd then press this button. It will add entire Departments to the SoA.
  • "Add Sub-Department to SoA". This button will add a Sub-Department (The third selection box) to the SoA.
    The third selection box? What third selection box?
    Well press the "Show Extra bits" button and you'll see.
    Select the Sub-Department(s) you want to add amd then press this button. It will add entire Sub-Departments to the SoA.
  • "Add Room to SoA". This button will add a Room (The fourth selection box) to the SoA. Select the Room(s) you want to add amd then press this button. It will add individual rooms to the SoA.
  • You can view the rooms that have been added by pressing the "View SoA" button.
  • If you think you've made a series of terrible mistakes and just want to start again, press the "Delete current SoA" button. This will delete all content in this projects SoA so you can start again.
    Doing this is irreversable and will erase all content in this projects SoA. Dont blame me if you do it by mistake

Time to export

Are you ready to export your beautifully curated set of rooms into a fully functional SoA in MS Excel?

To do so, press the "Create Soa" button. It may appear as if nothing is happening but in the background magic is happening.

Depending on where you are accessing this program you may be greeted with a file prompt. Choose where you want to save the SoA and what you want to call it. If you are in MS Access it will just go to your downloads folder

The SoA

Navigate to where you saved it and open it up. You should be presented with a fully functional SoA based on what you just created It should have formulas built in and have nicely formatted headers etc... For a run down on how it works, press the "SoA template Export" button at the top of this page.
